KulinerReview

The "KulinerReview" website is a platform designed to allow users to add and view restaurant reviews. With a simple layout dominated by green tones, the website features a top navigation menu that includes Home, Add Review, View Review, and Contact-Us. The main feature of this website is the review submission form, which consists of several input fields such as the user's name, restaurant name, star rating (ranging from 1 to 5), and a text box for writing their experience. Users can rate restaurants based on a scale from 1 star (Not Tasty) to 5 stars (Very Tasty). After filling out the form, they can submit their review by clicking the "Submit Review" button or view other reviews by selecting "View Review". With this concept, KulinerReview serves as a platform for food enthusiasts to share their dining experiences and help others discover the best places to eat.

HTML CSS Bootstrap MySQL

SamTravel

SAMTRAVEL is a web-based bus ticket booking platform that allows users to view, search, and add bus services with details such as name, service class, and ticket prices. This website uses Vue.js for the frontend and Laravel for the backend, with API communication handled via Axios. Key features include navigation to various pages such as services, users, bookings, payments, and routes, as well as a search system and the ability to add new buses. For future development, features like admin authentication, online ticket booking, digital payment integration, and service filtering and sorting can be added to enhance user-friendliness.
